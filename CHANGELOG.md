# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.4.0] - Unreleased
### Added
- Support for config server
- Add documentation for Notification Service

## [1.3.0] - 2025-12-19
### Added
- Prometheus metrics endpoint
- ELK stack integration
- Add support for notification type of CHANGE_VISIBILITY

## [1.2.0] - 2025-12-03
### Added
- More notifications
- Anti-spam features
- Healthcheck endpoint

## [1.1.0] - 2025-11-15
### Added
- Add multitenancy support

## [1.0.0] - 2025-11-07
### Added
- Initial release of Traversium Notification Service
- User authentication and authorization
- RESTful API for notification management
- Docker support with multi-platform builds (amd64, arm64)
- GitHub Actions CI/CD pipeline

### Security
- Implemented JWT-based authentication
